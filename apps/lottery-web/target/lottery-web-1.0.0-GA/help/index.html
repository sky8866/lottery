断规使用帮助

断规是系统经过多层分析而得出的结果，如果出来在断规里面的规律，表示这些规律要少用了，越是排在前面的越可以当成排除位排除了。 如图所示：



从图上得知95%与99%出现的规为9头与0头，所以我们1862期排除90头，然而1862期开的是1头，所以排除是对的。


从图上得知95%与99%出现的规为3百位与90，56，56百位，所以我们1862期排除90563百位，然而1862期开的是8百位，所以排除全部对的，很爽一下子排除5个数字。


1862期99%与95%这两组里没有出现规律，所以我们不能确定要排除十位。没办法了电脑帮不到，只能靠自己了。


从图上得知99%出现的规为1尾与27尾，而95%分组里没有出现规，不过没有关系，排名了127尾，知足了。最后1862期开的是6尾，所以排除是对的。
